CSS Properties Practice

In this exercise I am attempting to write some html, and format it using basic css text properties. Hopefully, doing this exercise will give me some practice typing, remembering some of the most common css properties, setting up files and using git.


Below are some of the most common css properties and a brief description of what they do...

  • Text-align

    By default your text will align to the left, just like it does in Word or Google Docs. Using text-align you can change it to align to the left, right, center or justify.

  • Font-style

    Font-style allows you to italicizeyour text. You can either use the words "oblique" or "italic", which will do the same thing.

  • Font-weight

    This is the font you will use for creating super light to bold text. You can use the values normal, bold, bolder and lighter or an number value from 100-900.

  • Text-transform

    You can use the text-transform property to make all of your text change using the following commands: capitlize, uppercase and lowercase.

  • Font-variant

    The most common use of font-variant is toput text into "small-caps". It is also short hand for other types of font variagtions especially numeric ones such as "slashed-zero".

  • Text-decoration

    This property lets you add lines over, under and through your text using the commands: overline, underline and line-through.

  • Line-height

    The default for line-height is to be the same as the font-size. You can adjust this by specifing specific pixels, ems, numbers or percentages.

  • Text-indent

    This sets the width of the indent at the beginning of a paragraph. It takes pixels, ems or percentages.

  • Word-spacing

    This sets the spacing between words, while letter-spacing sets the space between letters. It accepts pixels and ems as well as negative values.

That's enough for now! There are about 250 properties and I don't have time to list them all here.
